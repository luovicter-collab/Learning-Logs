# 📚 Learning Logs 学习记录

欢迎来到我的学习日志仓库！这里将持续记录我每日/每月的学习成长轨迹，无论是知识积累、技能实践，还是遇到的问题与思考，都将在此留下脚印。

---

## 📅 仓库结构

- 每个月一个 Markdown 文件，例如：
  - `2025-11 学习记录.md`
  - `2025-12 学习记录.md`
- 每个文件内，**按天记录**当日学习内容

---

## 📝 记录模板

每一天的记录大致包括：

```markdown
## YYYY/MM/DD 学习记录

### 今日目标
- 项目/知识点1
- 项目/知识点2

### 学习内容
1. 学到的具体内容
2. ...

### 遇到的问题和思考
1. 遇到的问题及解决方法
2. ...

### 今日收获和反思
- 收获/心得
